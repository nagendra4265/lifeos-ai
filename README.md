# LifeOS AI

A Flutter 3.35+ starter for an AI-powered personal life operating system.

## Phase 1 foundation

This scaffold introduces a clean, feature-first architecture with:

- Riverpod for state and dependency injection
- GoRouter for navigation
- Material 3 theme support with light/dark modes
- A modular folder structure for dashboard, assistant, notes, and auth features
- A widget test that validates the initial dashboard shell

## Proposed app structure

- lib/app
- lib/core
- lib/features
- lib/shared
